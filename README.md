# The Hotel

A modern web application for hotel management, featuring booking and interactive data visualization.  

**Deployed App:** [https://the-hotel-staff.netlify.app](https://the-hotel-staff.netlify.app)

## Technologies Used

- **Supabase** – Backend, database, and authentication  
- **TanStack React Query** – Data fetching and caching  
- **React Hook Form** – Form handling and validation  
- **Styled Components** – Scoped CSS styling  
- **Recharts** – Data visualization and charts  
- **date-fns** – Date formatting and utilities  
- **React Hot Toast** – Toast notifications  
- **React Icons** – Icon library for UI elements  
- **Vite** – Fast build and development tool
